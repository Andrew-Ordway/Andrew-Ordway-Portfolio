Seattle Skyline Hologram

For this project, I designed a hologram display system combining CAD modeling, mechanical design, and optical physics. Using CAD modeling software, I created two Seattle skyscrapers for a rotating platform display. The project used the Pepper's Ghost illusion with plexiglass positioned at 45 degrees to project the hologram against a laser-engraved Seattle Seahawks logo. Also, I integrated Arduino-controlled LED lighting to complete the system, showcasing my CAD skills and hands-on engineering abilities.

Mechanical Orrery Assembly

Mechanical Orrery Assembly by Andrew Ordway on Sketchfab

I designed this gravity-driven desktop orrery mechanism to demonstrate planetary motion through a counterbalanced lever system. This project allowed me to apply mechanical design principles and explore dynamic motion through weighted components.

Orrery Lever Arm And Platform by Andrew Ordway on Sketchfab

Orrery Fixed Gear Ratio by Andrew Ordway on Sketchfab

Key Features:

  • I developed a custom gear ring with accurate tooth geometry to translate the lever's arc into smooth rotation
  • I designed an organic lever arm with integrated counterweight system for continuous motion
  • I created a precision circular base housing with integrated mounting features and gear mechanism
  • I used a modular component approach for easy assembly and maintenance

Design Process:

  • I calculated the counterweight balance to achieve smooth, controlled motion
  • I incorporated smooth filleted transitions throughout for both aesthetics and stress distribution
  • I designed the base platform to support the dynamic forces during operation
  • I focused on a clean, minimal aesthetic that emphasizes the mechanical functionality

Arduino LED Matrix Pong Game

The Final Product

This project is a minimalist, hardware-based version of the classic Pong game, built using an Arduino UNO, a joystick module, and an 8x8 LED matrix display. The game runs entirely on Arduino, using no external screen or PC, and features interactive visual feedback for game events.

The player controls a paddle using an analog joystick to keep the ball in play. The game simulates basic physics, including bouncing off walls and paddle collision detection. When the player misses the ball, the game responds with a visual indicator (the entire screen lights up red) to signal game over before resetting.

Circuitry and Coding Components

Circuitry

The circuit uses a breadboard-based prototyping setup that organizes connections between the Arduino, joystick, and LED matrix:

Joystick Connections:

  • VCC (red) ‚Üí Arduino 5V via breadboard power rail
  • GND (yellow) ‚Üí Arduino GND via breadboard ground rail
  • VRx (purple) ‚Üí Arduino A0 for X-axis paddle control
  • VRy (teal) ‚Üí Arduino A1 (available for future features)

LED Matrix Connections:

  • Data (orange) ‚Üí Arduino Digital Pin 6
  • VCC (red) ‚Üí Arduino 5V
  • GND (white) ‚Üí Arduino GND

The joystick module is mounted directly on the breadboard for easy access during gameplay, while color-coded jumper wires provide clean signal routing between components. USB power supplies sufficient current for the LED matrix at the reduced brightness level used in gameplay.

Software Architecture

The game is programmed in C++ using the Arduino framework and Adafruit NeoPixel library for LED control. Key software features include:

Game State Management:

  • Real-time position tracking for paddle and ball
  • Collision detection algorithms for walls and paddle
  • Game-over state handling with visual feedback

Input Processing:

  • Analog-to-digital conversion of joystick position
  • Threshold-based paddle movement with deadzone handling
  • Smooth, responsive control at 10Hz update rate

Display Rendering:

  • Efficient pixel addressing using serpentine matrix mapping
  • Frame-based animation at configurable update rates

Physics Engine:

  • Wall collision detection and reflection
  • Paddle hit detection with dynamic angle adjustment based on contact point
  • Ball direction varies depending on which part of the paddle makes contact
