Seattle Skyline Hologram

For this project, I designed a hologram display system combining CAD modeling, mechanical design, and optical physics. Using CAD modeling software, I created two Seattle skyscrapers for a rotating platform display. The project used the Pepper's Ghost illusion with plexiglass positioned at 45 degrees to project the hologram against a laser-engraved Seattle Seahawks logo. Also, I integrated Arduino-controlled LED lighting to complete the system, showcasing my CAD skills and hands-on engineering abilities.

Arduino LED Matrix Pong Game

This project is a minimalist, hardware-based version of the classic Pong game, built using an Arduino UNO, a joystick module, and an 8x8 LED matrix display. The game runs entirely on Arduino, using no external screen or PC, and features interactive visual feedback for game events.

The player controls a paddle using an analog joystick to keep the ball in play. The game simulates basic physics, including bouncing off walls and paddle collision detection. When the player misses the ball, the game responds with a visual indicator (the entire screen lights up red) to signal game over before resetting.
